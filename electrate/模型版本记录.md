## 📊 模型版本 v1.0 - 基础模型

### 日期
2026-03-14

### 训练方法
- **算法**: Gradient Boosting Regressor
- **参数**: n_estimators=100, max_depth=5

### 特征
| 特征名 | 公式 | 说明 |
|--------|------|------|
| da_price | 直接使用 | 日前分时电价 |
| solar | AVG(solar_output) | 新能源平均出力 |
| load | AVG(total_load) | 统调负荷平均值 |
| gap | load - solar | 净负荷 |
| renewable_ratio | solar / (load + 1) | 新能源占比 |
| hour | 时段 (0-23) | 小时 |
| dow | dayofweek | 星期几 |
| is_weekend | dow >= 5 | 是否周末 |
| da_price_lag1 | shift(1) | 前一时段日前电价 |

### 数据
- **训练集**: 28天
- **测试集**: 7天
- **数据范围**: 2026-02-01 ~ 2026-03-10

### 性能
| 指标 | 数值 |
|------|------|
| MAE | 140.70 元/MWh |
| RMSE | 198.67 |
| R² | 0.071 |

### 说明
初始模型，使用基础特征

## 📊 模型版本 v3.2/3.3 - 加入供求关系

### 尝试
加入负载、发电等供求关系特征，但提升有限

### 性能
| 版本 | MAE | 说明 |
|------|-----|------|
| v3.1 | 70.19 | 漫湾厂为主 |
| v3.3 | 71.96 | +供求关系 |

### 结论
供求关系特征贡献较小，漫湾厂数据已高度相关

### 日期
2026-03-14

### 训练方法
- **算法**: Gradient Boosting Regressor
- **参数**: n_estimators=200, max_depth=6

### 特征
| 特征名 | 公式 | 说明 |
|--------|------|------|
| rt_price | 实时电价 | 目标变量 |
| generation | 发电总出力 | 从hourly_generation |
| nonmarket | 非市场化机组出力 | 从hourly_nonmarket |
| manwan_price | 漫湾厂.500kV#1M | **新增** 漫湾厂节点电价 |
| manwan_ratio | manwan_price / rt_price | 漫湾厂占比 |
| nonmarket_ratio | nonmarket / (generation+1) | 非市场化比例 |
| gen_nonm_ratio | generation / (nonmarket+1) | 发电/非市场化比 |
| hour | 时段 | 小时 |
| dow | dayofweek | 星期几 |
| is_weekend | dow >= 5 | 是否周末 |
| rt_price_lag1 | shift(1) | 前一时段电价 |

### 数据
- **训练集**: 644条
- **测试集**: 161条
- **日期范围**: 38天

### 性能
| 指标 | 数值 |
|------|------|
| MAE | **18.09 元/MWh** |
| RMSE | 50.81 |
| R² | 0.939 |

### 特征重要性
| 特征 | 重要性 |
|------|--------|
| rt_price | 99.3% |
| nonmarket_ratio | 0.3% |
| manwan_price | 0.2% |
| hour | 0.1% |

### 说明
增加漫湾厂水电节点电价作为特征，MAE从100.96降至18.09，**提升82.1%**

### ⚠️ 注意
当前模型使用实时电价作为主要特征，接近"用今天预测今天"的模式。实际预测时需要用日前电价替代。
