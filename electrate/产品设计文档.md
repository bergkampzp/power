## 一、产品概述

### 1.1 产品背景

面向水电站的电力价格预测工具，帮助水电站运营者：
- 预测未来电价走势
- 优化中长期合约签订时机
- 现货分时套利决策
- 制定发电计划

### 1.2 目标用户

| 用户类型 | 需求 |
|----------|------|
| 小水电站 | 日常电价参考 |
| 中型水电站 | 交易策略建议 |
| 大型水电厂 | 负荷分配优化 |

### 1.3 核心价值

- **降低风险**: 预测电价走势，避免低价售电
- **提升收益**: 优化交易时机，提高售电均价
- **辅助决策**: 提供数据支撑的交易建议

## 四、数据展示页面 (V1.2)

### 4.1 页面概述

独立数据展示页面，直观展示电力市场各类数据。

**访问地址**: http://localhost:5175/

**技术栈**:
- 纯HTML + JavaScript
- ECharts 图表库 (CDN)
- Python HTTP Server 服务
- 端口: 5175 (独立服务)

### 4.2 布局结构

```
┌─────────────────────────────────────────────────┐
│  顶部：标题 + 日期选择器                         │
├────────────┬────────────────────────────────────┤
│            │                                    │
│  左侧      │  右侧：田字格展示 (2x2)           │
│  4个分类   │  ┌─────────────┬─────────────┐    │
│  数据列表  │  │ 负载数据    │ 新能源数据  │    │
│            │  ├─────────────┼─────────────┤    │
│            │  │ 水电数据    │ 电价数据    │    │
│            │  └─────────────┴─────────────┘    │
└────────────┴────────────────────────────────────┘
```

### 4.3 左侧分类菜单 (4大类)

| 分类 | 颜色 | 数据项 |
|------|------|--------|
| 📊 负载相关 | 红色 #f5222d | 日前用电需求(分时)、系统负荷(分时) |
| 🌞 新能源相关 | 绿色 #52c41a | 光伏出力(分时)、新能源总出力(分时) |
| 💧 水电相关 | 蓝色 #1890ff | 水电出力、水电利用率 |
| 💰 电价相关 | 紫色 #722ed1 | 日前节点电价(分时)、日前用电侧电价(分时) |

### 4.4 功能特点

1. **日期选择器**: 顶部选择日期，右侧4个区域同步更新
2. **田字格展示**: 2×2布局，分别展示4类数据
3. **分时曲线**: 每个数据项展示24小时分时曲线
4. **点击切换**: 点击左侧数据项，右侧对应区域显示该数据
5. **实时统计**: 底部显示均值、最大值、最小值

### 4.5 数据源

| 数据项 | 来源表 | 分时 |
|--------|--------|------|
| 日前用电需求 | day_ahead_demand | ✅ 24点 |
| 系统负荷 | grid_load_overview | ✅ 24点 |
| 光伏出力 | renewable_output | 推算 |
| 日前节点电价 | day_ahead_node_price | 映射 |

## 五、数据库表结构

### 5.1 核心数据表

| 表名 | 用途 | 关键字段 |
|------|------|----------|
| renewable_output | 新能源出力 | solar_output, wind_output, total_renewable |
| grid_load_overview | 电网负荷 | total_load, peak_load, valley_load |
| day_ahead_demand | 日前需求 | demand (24点分时) |
| day_ahead_node_price | 日前节点电价 | avg_price, min_price, max_price |
| realtime_node_price | 实时节点电价 | avg_price, min_price, max_price |
| hydro_output | 水电出力 | output, capacity, utilization_rate |

### 5.2 当前数据规模

- **数据范围**: 2026-02-01 ~ 2026-03-10
- **有效天数**: 36天
- **分时数据**: 日前需求24点/天
