## 三、数据流程

### 3.1 数据采集

```python
# 1. 电价爬取
scraper.py → 电力市场网站 → Excel文件

# 2. 气象获取
weather_api.py → Open-Meteo → JSON

# 3. 手动导入
import_data.py → Excel → SQLite
```

### 3.2 数据处理

```python
# 1. 数据清洗
df_clean = df.dropna()  # 去除缺失
df_clean = df[df > 0]   # 去除异常

# 2. 特征工程
features = {
    'time': dayofweek, hour, month,
    'lag': lag_1, lag_2, lag_7,
    'ma': ma_3, ma_7, ma_14,
    'external': load, solar, hydro
}

# 3. 数据存储
SQLite → power_market.db
```

### 3.3 预测流程

```python
# 1. 输入特征
features = load_features(date)

# 2. 模型预测
prediction = model.predict(features)

# 3. 输出结果
return {
    'date': date,
    'predicted_price': price,
    'confidence': confidence,
    'reason': reason
}
```

## 四、数据库设计

### 4.1 核心表

```sql
-- 日前节点电价
CREATE TABLE day_ahead_node_price (
    id INTEGER PRIMARY KEY,
    trade_date TEXT,
    region TEXT,
    node_name TEXT,
    avg_price REAL,
    min_price REAL,
    max_price REAL
);

-- 实时节点电价
CREATE TABLE realtime_node_price (
    id INTEGER PRIMARY KEY,
    trade_date TEXT,
    region TEXT,
    node_name TEXT,
    avg_price REAL,
    min_price REAL,
    max_price REAL
);

-- 用电侧数据
CREATE TABLE day_ahead_demand (
    id INTEGER PRIMARY KEY,
    trade_date TEXT,
    period TEXT,
    demand REAL,
    price REAL
);

-- 负荷预测
CREATE TABLE load_forecast (
    id INTEGER PRIMARY KEY,
    trade_date TEXT,
    period TEXT,
    forecast_load REAL,
    actual_load REAL
);

-- 新能源出力
CREATE TABLE renewable_output (
    id INTEGER PRIMARY KEY,
    trade_date TEXT,
    period TEXT,
    solar_output REAL,
    wind_output REAL
);

-- 水电出力
CREATE TABLE hydro_output (
    id INTEGER PRIMARY KEY,
    trade_date TEXT,
    period TEXT,
    plant_name TEXT,
    output REAL
);

-- 系统备用
CREATE TABLE system_reserve (
    id INTEGER PRIMARY KEY,
    trade_date TEXT,
    period TEXT,
    positive_reserve REAL,
    negative_reserve REAL
);
```

## 五、API设计

### 5.1 接口列表

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/price/daily` | GET | 获取日均价 |
| `/api/price/ realtime` | GET | 获取实时电价 |
| `/api/price/node/:name` | GET | 节点电价 |
| `/api/predict` | POST | 电价预测 |
| `/api/predict/weekly` | GET | 周预测 |
| `/api/data/export` | GET | 数据导出 |
| `/api/advice` | GET | 交易建议 |

### 5.2 预测接口示例

```bash
# 请求
POST /api/predict
{
    "date": "2026-03-15",
    "features": {
        "lag_1": 250.5,
        "lag_7": 240.2,
        "dayofweek": 6,
        "load": 15000
    }
}

# 响应
{
    "success": true,
    "prediction": {
        "date": "2026-03-15",
        "price": 258.3,
        "confidence": 0.85,
        "range": [245.0, 271.6]
    }
}
```
